# Provenant Nix Flake

A [Nix flake](https://nix.dev/concepts/flakes) providing [provenant](https://github.com/mstykow/provenant) as a package, app, and overlay.

```bash
nix run .#                          # run provenant
nix build .#                        # build the package
```
